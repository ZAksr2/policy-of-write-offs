Політика безпечного списування

Ця Політика безпечного списування (далі — "Політика") регулює умови, на яких особа Д. надає свої виконані завдання іншим учням (далі — "Користувачі") для особистого використання в рамках навчального процесу. Використовуючи роботи особи Д., Користувачі погоджуються з умовами цієї Політики.

1. Надання доступу до робіт

1.1. Особа Д. на добровільній основі надає свої виконані завдання Користувачам для таких цілей:

  • для перевірки власної роботи,
  • для навчання,
  • для списування (за бажанням Користувачів).

1.2. Роботи особи Д. надаються на тимчасовій основі й виключно для особистого користування Користувачами.

2. Обмеження розголошення

2.1. Користувачі зобов’язуються не розголошувати факт отримання робіт від особи Д. третім особам, зокрема іншим учням, викладачам або адміністрації навчального закладу.

2.2. Усі роботи, надані особою Д., є конфіденційними та призначені виключно для індивідуального використання Користувачами.

3. Відповідальність за правильність роботи

3.1. Особа Д. не несе відповідальності за правильність виконання завдань, які надаються Користувачам. Користувачі погоджуються, що отримані матеріали можуть містити помилки або неточності.

3.2. У разі, якщо Користувачі списують роботи особи Д., і ці роботи містять помилки, Користувачі не мають права звинувачувати особу Д. у дезінформації або наданні неправильної інформації.

3.3. Користувачі беруть на себе повну відповідальність за використання робіт особи Д. і погоджуються, що вони роблять це на власний ризик.

4. Використання робіт

4.1. Роботи, надані особою Д., можуть бути використані Користувачами лише для особистих потреб, як то для перевірки власних знань, самостійного навчання або списування.

4.2. Використання робіт особи Д. з іншою метою (зокрема, для передачі іншим учням або використання в комерційних цілях) забороняється.

5. Заключні положення

5.1. Ця Політика є добровільною угодою між особою Д. і Користувачами.

5.2. Особа Д. залишає за собою право в будь-який момент припинити надання робіт Користувачам без попередження.

5.3. У разі порушення умов цієї Політики Користувачами, особа Д. має право відмовити їм у подальшому наданні своїх робіт.

Погоджуючись із цією Політикою, Користувачі підтверджують своє розуміння та прийняття всіх умов, зазначених вище.
